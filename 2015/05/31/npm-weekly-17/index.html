npm 週報 #17

你希望可以收到 npm 最新更新的消息嗎？

一個更好的命令列工具

上個禮拜我們展示如何建置一個簡單的命令列工具給你看。這個禮拜，我們告訴你如何改進你的工具。

  • 增加子命令到命令列工具
  • 讓命令列工具變成可設定的

npmHumans++

我們增加了一個新人，現在在 registry 團隊。Ryan Day 將會加入 CJ、Ben 和 Maciej 來確保 registry 可以跟上不斷上升的流量。

npmHumans.speak()

說到曾經爆衝的流量，CJ 禮拜五將會在 Web Rebels 和禮拜六在 NodeConf One-Shot Oslo 談論團隊如何讓 registry 擴展。

如果你對 在 npm@3 可以期待什麼 有興趣，可以參考 Rebecca’s 在 Boston Ember 聚會的談話。

還有 Raquel 用播客談到關於她成為一個 Web 開發者的旅程在 很開心，Isaac 寄了 Email 給我。

nodeschool 國際日

想要學 Node？這個星期六將會有 NodeSchools 在世界各地 舉辦。

如果你在學找如何學習 (或教) npm，可以嘗試 how-to-npm workshop。

視覺化的 spdx 支援

在上個禮拜的 weekly，我們談到 SPDX。隨著 SPDX License Expression Syntax，你可簡單的用工具找出你的程式碼的許可方式。這對企業使用者非常好。

如果你想知道哪個套件有可解析的許可證字串 (還有哪個沒有)，可以查看 Kyle Mitchell 的新網站，npm1K。
