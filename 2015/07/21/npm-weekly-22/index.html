npm 週報 #22 : 年度的改變遊戲規則者

年度的改變遊戲規則者

我們在 the net awards 年度的改變遊戲規則者 的名單上！感謝每個提名我們的人！

nodeup 上有 npm 3 的背景故事

想知道我們是如何走到 npm 3？Rebecca、Forrest 和 Kat 在 NodeUp Podcast 上給你完整的背景故事和最新消息。

npm 3 beta 測試者：請升級到 3.1.2！

原來是有一個 bug 在 npm@3.1.1 (這使它難以在本地下載模組)，所以昨天 Rebecca 跳脫週期釋出一個新的版本。你可以藉由 npm install -g npm@v3.x-next 升級。

談到升級 (windows 使用者)…

如果你想要在 windows 上升級 npm，請查看 npm-windows-upgrade。

在 Windows 上升級 npm 需要幾個手動步驟以確保 PowerShell/CMD 找得到新版 npm。這個是 Microsoft DX 工程師為了 npm 和 Node 用 :heart: 做的小工具，把過程簡化成一個簡單指令。

對 prepublish 感到疑惑？

Stephan Bönnemann 解釋了 圍繞 prepublish 生命週期腳本的疑惑和為什麼它的奇怪行為可能其實比它看起來的更有幫助。

nodeschool 奧克蘭

想要學習如何寫 Node？這週末在 NodeSchool 奧克蘭來 npm 辦公室與 其他學習者和導師一起廝混。空間有限，所以請一定要註冊。

如果你想要從家裡跟隨，你可以查看所有的工作坊並藉由 npm 下載它們。
